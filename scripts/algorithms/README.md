# Algorithm Scripts

This directory contains scripts for updating and maintaining the packing algorithm implementations and calculations.

## Scripts

- `update_algo.py` - Updates main algorithm implementation in Rust
- `move_calc.py` - Handles movement calculations
- `replace_table.py` - Replaces table data structures
- `remove_auto_calc.py` - Removes automatic calculations
- `move_bounds.py` - Manages movement boundaries

## Usage

Run these scripts from the project root directory:

```bash
python scripts/algorithms/update_algo.py
python scripts/algorithms/move_calc.py
```

## Purpose

These scripts automate updates to the packing engine algorithm, which is responsible for optimal 2D cutting optimization and part placement calculations.

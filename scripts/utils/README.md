# Utility Scripts

This directory contains various utility scripts for maintenance, configuration, and feature additions.

## Scripts

- `fix_worker.py` - Fixes worker thread and Web Worker related issues
- `check_ui.py` - Validates UI component integrity
- `update_drawer_offset.py` - Updates drawer offset calculations
- `update_drawer_slider.py` - Updates drawer slider implementation
- `add_pdf_export.py` - Adds PDF export functionality

## Usage

Run these scripts from the project root directory:

```bash
python scripts/utils/check_ui.py
python scripts/utils/add_pdf_export.py
```

## Purpose

These scripts help maintain and extend the application with new features and bug fixes.

# Yaseen Smart Design & Packing Application

A high-performance React + Vite web application seamlessly wrapped with Capacitor to deliver a native Android experience. This application is engineered for automated structural layouts, smart template-based frameworks, and multi-material packing optimization.

---

## 🚀 Tech Stack
* **Frontend Framework:** React, Vite, TypeScript
* **Styling:** Tailwind CSS
* **Native Runtime Container:** Capacitor Core & Android CLI
* **Automation & Compilation:** Gradle Build System

---

## 🛠️ Core Features
* **Multi-Material Packing Support:** Advanced optimization algorithms to calculate, align, and pack multi-material components cleanly.
* **Template-Based Designing:** Dynamic framework generation where inputting custom dimensions automatically structures modular panels and layout partitions.
* **Mobile Terminal Optimization:** Responsive and lightweight UI designed to run smoothly on Android environments.

---

## 📦 Production Build & APK Generation Flow

Agar future me aapko application me koi bhi modification karke nayi upgrade APK generate karni ho, toh terminal me yeh standard steps follow karein:

### 1. Build the Production Web Assets
Compile the updated TypeScript and React code into optimized web assets:
```bash
npm run build
{
  "appId": "com.yaseen.app",
  "appName": "Yaseen App",
  "webDir": "dist"
}
npx @capacitor/cli sync
cd android && ./gradlew assembleDebug
Your newly compiled application package will be available at:
android/app/build/outputs/apk/debug/app-debug.apk
​📂 Project Architecture
​src/ - Main development directory containing all core code, components, and layout calculation logic.
​dist/ - Distribution directory containing optimized production-ready web assets generated by Vite.
​android/ - Core native Android studio project folder containing Gradle configurations and platform assets.
​capacitor.config.json - Native runtime configuration file mapping the web assets to the Android container.
